
# Request: Periodontal/Preventive Form Enhancements (Implementation Spec)

## Goal
Update the periodontal and hygiene documentation form so it supports the clinic's required diagnostic wording, treatment options, OHE options, and follow-up workflow.

## Scope
This request covers:
- Periodontal status section
- Caries risk section
- Treatment done today section
- Next appointment section
- OHE and recommendations
- EOE/IOE quick-select findings
- Follow-up disposition labels

## Functional Requirements

### 1) Periodontal Status
Add explicit selectors for:
- Activity status: `Active`, `Stable`
- Disease type: `Periodontitis`, `Gingivitis`

Replace the current stage-only wording with these severity/stage options:
- `Slight Periodontitis Stage I`
- `Moderate Periodontitis Stage II`
- `Severe Periodontitis Stage III`
- `Severe Periodontitis Stage IV`

Replace generic grade labels with:
- `Grade A slow rate of progression`
- `Grade B moderate rate of progression`
- `Grade C rapid rate of progression`

### 2) Treatments
Add treatment option:
- `Hand and power instrumentation`

For `Hand and power instrumentation`, add device sub-options:
- `Cavitron`
- `Piezo`

For instrumentation, add area/location sub-options:
- `Q1`, `Q2`, `Q3`, `Q4`
- `Maxilla`, `Mandible`
- `Sextant 1`, `Sextant 2`, `Sextant 3`, `Sextant 4`, `Sextant 5`, `Sextant 6`

Add treatment option:
- `Ipana 5% NaF varnish application`

### 3) OHE Topics
Add selectable OHE topics:
- `Caries theory`
- `Caries risk factors`
- `Bass brushing`
- `C-shape flossing technique`
- `Sulcabrush and interdental brush technique`
- `Review benefits of Prevident or Opti-Rinse`
- `Periodontitis theory and risk factors`
- `Importance of maintaining a 4-month hygiene interval`

### 4) Recommendations
Add selectable recommendations:
- `High fluoride toothpaste (Prevident 5000)`
- `Mouthwash (0.05% X-PUR Opti-Rinse)`
- `Salt water rinse for 2-3 days`

### 5) Treatment Done Today
Add the following as selectable items:
- `Med/dent history update`
- `EOE/IOE`
- `Gingival assessments`
- `Calculus index`
- `Caries risk`
- `Nutrition score`
- `Periodontal risk assessment`
- `Spot probing`
- `Full mouth probing`
- `Full mouth hand and power (Piezo) instrumentation`

### 6) Next Appointment
Provide the same selectable items as `Treatment Done Today` for `Next Appointment`.

### 7) Follow-up/Disposition Terminology
Add final disposition options:
- `DH Re-eval at 4-6 weeks`
- `DH Re-care at 3-4 months interval`

Terminology rule:
- Do not use `recall` for hygiene re-care.
- `Recall` is reserved for DDS workflows.

### 8) Caries Risk Section
Add a dedicated `Caries risk` section (separate, explicit section in the form).

### 9) EOE/IOE Quick Select (Within Normal Limits)
Add an auto-select option to mark `Within Normal Limits` for EOE/IOE.

Include selectable findings for:
- `Asymptomatic click on opening/closing` with laterality options: `Bilateral`, `Left`, `Right`
- `Coated tongue`
- `Fissured tongue`
- `Bilateral linea alba`
- `Palatine torus at midline` with severity options: `Slight`, `Prominent`

## UI/Behavior Notes
- Keep all new items as structured selectable options (not free-text only).
- Preserve ability to combine multiple selections in each section.
- Ensure wording in the UI matches labels above exactly.
- Selecting `Within Normal Limits` must still allow documentation entry (editable notes/observations remain enabled).

## Acceptance Criteria
- All options listed in this document are available in the appropriate sections.
- `Treatment Done Today` and `Next Appointment` contain matching option sets.
- Periodontal status uses the requested activity/disease/severity-grade wording.
- EOE/IOE includes a working `Within Normal Limits` auto-select option and listed findings.
- `Within Normal Limits` does not block writing notes/observations in EOE/IOE.
- Follow-up labels use `DH Re-care` and avoid `recall` for hygiene.
- Caries risk appears as its own section in the form.
